## Quick Start

```bash
# Clone the repository
git clone https://github.com/eduardg7/openclaw-os.git
cd openclaw-os

# Install dependencies
npm install

# Start the application
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and follow the onboarding wizard!

## Why OpenClaw OS?

Managing AI agents shouldn't be complicated. OpenClaw OS provides a unified dashboard to:

- **Monitor** all your agents in real-time
- **Track** tasks and sessions across projects
- **Configure** settings with visual feedback
- **Deploy** with one command using Docker

Built for the OpenClaw community, open source forever.
